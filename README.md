# Company Directory App

A full-stack MERN application for managing company records with authentication and CRUD operations.

## Features

* Admin Login Authentication
* Protected Routes
* Add Company
* View Company Details
* Edit Company
* Delete Company
* Delete All Companies
* Search Companies
* Pagination
* Company Count
* Loading Spinner
* Responsive Tailwind CSS UI

## Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Axios
* React Router DOM

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
